#Imersão Dados Alura

Projeto realizado durante a Imersão Dados da Alura, onde explorei conceitos de análise de dados usando Python.

Sobre o projeto: 
Este projeto tem como objetivo analisar um conjunto de dados públicos para extrair insights relevantes utilizando Python e bibliotecas como Pandas e Matplotlib. Pratiquei manipulação de dados, limpeza, visualização e geração de gráficos.

Tecnologias utilizadas:
Python  
Pandas  
Matplotlib  
Colab Notebook

Conteúdo do projeto:
`notebook.ipynb` – Notebook com análise exploratória, visualizações e comentários  
`dados/` – Pasta com os datasets utilizados 
`README.md` – Documentação do projeto

Como rodar o projeto:
1. Clone o repositório:  
   ```bash
   git clone https://github.com/vanscopel/imersao-dados-alura.git
2.Abra o notebook pelo Google Colab:
Acesse Google Colab
Clique em “File” > “Open notebook” > aba “GitHub”
Cole a URL do repositório ou busque pelo seu usuário e selecione o notebook .ipynb
3.Execute as células para ver a análise.
